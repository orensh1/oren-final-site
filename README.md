# Oren Shamir - Landing Pages

This is the source code for Oren Shamir's landing page website, built with React, TypeScript, and Tailwind CSS.

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)

### Installation

1.  Clone the repository or download the files.
2.  Install dependencies:

    ```bash
    npm install
    ```

### Development

To start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to see the app.

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` folder.

### Project Structure

- `src/` - Source code (React components, styles hooks).
- `src/components/` - UI components (Hero, Navbar, etc.).
- `tailwind.config.js` - Tailwind CSS configuration.
- `vite.config.ts` - Vite configuration.
